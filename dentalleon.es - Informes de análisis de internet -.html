Integrations

We provide step by step guides explaining how to embed the tracking code for some popular CMS and website builders:

WordPress | Squarespace | Wix | SharePoint | Joomla | Shopify | Google Tag Manager

En la mayoría de sitos web, blogs, CMS, etc. puede utilizar plugins precodificados para que realicen el trabajo técnico por usted. (Vea nuestralista de plugins para integrar Matomo.)

Si no existe ningún plugin, siempre puede editar las plantillas de su sitio web y agregar código JavaScript de seguimiento en la etiqueta </head>, la cual suele estar definida en "header.php", "header.tpl" o ficheros similares.

Código de seguimiento

Asegúrese que este código se encuentre en cada página de su sitio web. Recomendamos copiarlo inmediatamente antes de cerrar la etiqueta </head>

<!-- Matomo -->
<script>
  var _paq = window._paq = window._paq || [];
  /* tracker methods like "setCustomDimension" should be called before "trackPageView" */
  _paq.push(['trackPageView']);
  _paq.push(['enableLinkTracking']);
  (function() {
    var u="https://dentalleon.matomo.cloud/";
    _paq.push(['setTrackerUrl', u+'matomo.php']);
    _paq.push(['setSiteId', '1']);
    var d=document, g=d.createElement('script'), s=d.getElementsByTagName('script')[0];
    g.async=true; g.src='//cdn.matomo.cloud/dentalleon.matomo.cloud/matomo.js'; s.parentNode.insertBefore(g,s);
  })();
</script>
<!-- End Matomo Code -->

Si desea hacer más que rastrear vistas de páginas, revise la Documentación de Javascript de Matomo acerca de las funciones disponibles. Estas puede rastrear objetivos, variables personalizadas, órdenes de comercio electrónico, carritos abandonados y más.
